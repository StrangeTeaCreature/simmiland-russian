# Simmiland — Русификатор

Полный русский перевод для игры [Simmiland](https://store.steampowered.com/app/932850/Simmiland/) (Sokpop Collective).

> **752 из 753** строк переведены. Исправлено **~40 орфографических ошибок** оригинального community-перевода. Добавлена кириллица во все 25 шрифтов игры. Расширены карточки для длинного русского текста.

## Почему нельзя просто заменить CSV?

Игра использует движок GameMaker Studio. Все шрифты хранятся внутри файла `data.win` и **не содержат кириллических символов** — только латиницу, CJK и спецсимволы. Если просто подкинуть CSV с русским переводом, вместо текста будет пустота.

Наш мод решает это: **кириллические буквы нарисованы и добавлены во все 25 шрифтов игры** прямо в `data.win`.

## Установка (простой способ)

1. Скачайте архив **`simmiland-russian-v1.0.zip`** из [Releases](../../releases/latest)
2. Найдите папку с игрой:
   - **Steam**: ПКМ на Simmiland в библиотеке → Управление → Просмотреть локальные файлы
3. **Сделайте бэкап** — скопируйте `data.win` и `simmiland_localization.csv` куда-нибудь
4. Замените оба файла из архива в папку игры
5. Запустите игру → Settings → выберите **Русский** (на месте бывшего "español")

## Установка (для продвинутых — из скриптов)

Если вы хотите применить русификацию к другой версии игры или модифицировать перевод — можно собрать мод самостоятельно.

### Требования

- Python 3.x + [Pillow](https://pypi.org/project/Pillow/) (`pip install pillow`)
- [UndertaleModTool CLI](https://github.com/UnderminersTeam/UndertaleModTool/releases)
- Шрифт **Consolas** (есть в Windows по умолчанию)

### Порядок

```
1. Сделайте бэкап data.win и simmiland_localization.csv

2. Исправить перевод и скопировать русский текст в слот spanish:
   python scripts/fix_localization.py

3. Экспортировать шрифты из data.win:
   UndertaleModCli.exe load data.win -s scripts/export_fonts.csx

4. Нарисовать кириллицу на текстурах шрифтов:
   python scripts/add_cyrillic_to_textures.py

5. Импортировать модифицированные шрифты обратно:
   UndertaleModCli.exe load data.win -s scripts/import_cyrillic.csx -o data.win

6. Расширить карточки для длинного текста:
   UndertaleModCli.exe load data.win -s scripts/widen_cards.csx -o data.win

7. Готово! Запустите игру и выберите язык "Русский".
```

> Опционально: `scripts/reduce_shifts.py` сжимает межбуквенное расстояние (если текст не помещается). По умолчанию не нужен — вместо этого расширены карточки.

## Что было сделано

### Шрифты
Все 66 кириллических символов (А-Я, а-я, Ё, ё) нарисованы для каждого из 25 шрифтов игры. Для рисования использован системный шрифт Consolas с индивидуальным подбором размера. Вертикальное выравнивание рассчитано по образцу оригинальных латинских букв.

### Перевод
Русский перевод (из [community-документа](https://docs.google.com/spreadsheets/d/1l6eTaGDBhoGUKfgmtFn8r8aVBtaFL_HfYIZp9S6RKcQ/)) скопирован в слот испанского языка, т.к. добавление нового языка потребовало бы модификации кода меню. Исправлено ~40 опечаток (`используеться` → `используется`, `братарейка` → `батарейка`, `землятрясение` → `землетрясение` и др.).

### Карточки
Код объекта `oCard` модифицирован: поверхность карточки расширена на 80px, содержимое сдвинуто на 40px — длинный русский текст больше не обрезается.

## Известные ограничения

- **Качество шрифтов**: кириллица нарисована через Consolas/Pillow — на мелких шрифтах может быть слегка размыто
- **Испанский язык заменён**: русский занимает слот `español`. Одновременно оба языка недоступны
- **Длинные слова**: самые длинные слова (`землетрясение`, `человечество`) могут слегка выезжать за край карточки — это сделано намеренно, чтобы текст оставался читаемым
- **Перевод неофициальный**: орфография исправлена, но смысловые неточности могут остаться

## Совместимость

- Simmiland Build 7533614 (проверено)
- Должно работать с версиями, использующими тот же формат `data.win`

## Скрипты

| Файл | Описание |
|------|----------|
| `scripts/fix_localization.py` | Исправление опечаток + копирование русского в spanish |
| `scripts/export_fonts.csx` | Экспорт шрифтов из data.win (UndertaleModTool) |
| `scripts/add_cyrillic_to_textures.py` | Рисование кириллических букв на текстурах |
| `scripts/import_cyrillic.csx` | Импорт модифицированных шрифтов обратно |
| `scripts/widen_cards.csx` | Расширение карточек для длинного текста |
| `scripts/reduce_shifts.py` | (опционально) Сжатие межбуквенного расстояния |
