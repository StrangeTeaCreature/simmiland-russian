# Simmiland — Русификатор

Полный русский перевод для игры [Simmiland](https://store.steampowered.com/app/932850/Simmiland/).

## Что это

Файл  с полным переводом игры на русский язык.
- **752 из 753** текстовых строк переведены на русский
- Перевод основан на [официальном community-документе](https://docs.google.com/spreadsheets/d/1l6eTaGDBhoGUKfgmtFn8r8aVBtaFL_HfYIZp9S6RKcQ/) переводчиков Simmiland

## Установка

1. Скачайте файл  из этого репозитория
2. Найдите папку с игрой:
   - **Steam**: правый клик на Simmiland в библиотеке → Управление → Просмотреть локальные файлы
   - **Другое**: найдите папку где лежит 
3. **Сделайте бэкап** — скопируйте оригинальный  куда-нибудь (на всякий случай)
4. Замените  в папке игры на скачанный
5. Запустите игру → Settings → выберите **Русский**
